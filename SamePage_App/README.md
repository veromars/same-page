# Same Page (p.2) — Premium Social Discovery Web App

A high-fidelity social networking prototype designed for meaningful discovery through shared stories and curated meetups. Built with modern web standards and premium aesthetics.

## ✨ Key Features
- **Notebook-Themed Discovery**: Reveal personal stories through a tactile, chapter-based UI.
- **High-Fidelity Meetup Feed**: Discover and save community events with rich metadata and social proof.
- **Dynamic Profile Engagement**: A reward-based system where your activity (writing answers/completing chapters) unlocks more benefits.
- **Saved Collections**: Real-time synchronization between event discovery and personal bookmark lists.

## 🛠 Tech Stack
- **Structure**: Semantic HTML5
- **Logic**: Vanilla JavaScript (Modern ES6+)
- **Styling**: Vanilla CSS (Premium design system with Glassmorphism)
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Typography**: Outfit, Pretendard, Style Script

## 🚀 How to Run Locally
1. Clone this repository or download the ZIP.
2. Open `index.html` in any modern web browser.
   - *Note: For the best experience, we recommend using a simple local server (like Live Server in VS Code) to handle asset loading smoothly.*

## 📦 Deployment Guide (GitHub Pages)
This app is static and ready to be hosted on **GitHub Pages** for free:
1. Create a new repository on GitHub.
2. Push the contents of the `SamePage_App` folder to the `main` branch.
3. Go to **Settings** > **Pages**.
4. Select `main` branch and `/ (root)` folder as the source.
5. Click **Save**. Your site will be live at `https://<username>.github.io/<repository-name>/`.
